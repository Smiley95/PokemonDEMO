# PokemonDEMO

## Overview
This code is based on this project (https://github.com/llSourcell/Pokemon_GAN/) It's aim is to use a WGAN to create new kinds of Pokemon. 

## Dependencies (pip install) 
```
cv2
tensorflow( >=1.0)
scipy
numpy
imagio
```
## Usage
```
python resize.py
python RGBA2RGB.py
python PokemonDemo.py
```
